# Listing Image Template Brief

> Use Figma as the source of truth. Export PNGs at **2000x2000** (Etsy's
> recommended square aspect) for the hero and detail images.

## Style System
- **Palette:** deep pasture green (#1F3B24), warm lumber tan (#C89B61), cream paper (#F7F3E8), charcoal (#252525), rust brown (#8A4B2F), hardware-cloth gray (#7D8178).
- **Font:** Sans-serif headings (e.g., Inter Bold), serif body for callouts (e.g., Source Serif).
- **Iconography:** simple line icons; consistent stroke width.
- **Tagline footer:** "PastureFrame Plans — Designed by Ashton Aschenbrener" small caps, bottom-right.

## 10 Image Slots
| # | Purpose | Required Elements |
|---|---------|-------------------|
| 1 | Hero | Bold "DIGITAL PLANS — NO PHYSICAL COOP SHIPPED" overlay, product name, dimensions teaser. |
| 2 | What's Included | Checklist of files (PDF, materials list, cut list, etc.). |
| 3 | Top-View Plan | 10' x 10' run, 4' x 10' upper coop. |
| 4 | Side-View Plan | Lower run height 48"-54", upper coop wall height 42"-48". |
| 5 | Frame / Exploded View | Skids, base frame, upper coop frame, roof framing. |
| 6 | Wheel / Skid Detail | 4x4 skids, axle or hub/spindle wheels, tow tongue. |
| 7 | Layer-Hen Setup | 3 nest boxes, removable roosts, pop door. |
| 8 | Broiler Setup | Open lower run, feeders, waterers, shade, frequent moves. |
| 9 | What's Included | PDF, material list, cut list, 15-step build guide. |
| 10 | Safety / Disclaimer | DIY plans, verify local conditions, no physical coop shipped. |

## Required Compliance Overlays
- **Slot 1 (hero) MUST clearly say "DIGITAL PLANS"** in the top quartile.
- **Slot 8 MUST include the disclaimer text card.**
- **No stock photos** as the primary listing image (Etsy 2026 rule).
- If AI was used to generate any image, add small "AI-assisted" mark in
  bottom-right and disclose in the listing description.
