# PastureFrame Chicken Coop Final Plan Package

This ZIP contains the chicken coop build-spec package requested for the 10 ft x 10 ft mobile coop project.

## Locked decisions

- Mobility system: 4x6 skids + axle-mounted pneumatic wheels + tow tongue.
- Lower run height: 48 in for the mobile version.
- Upper coop: 4 ft x 10 ft elevated house.
- Publishing assumption: sell from CAD/plans first unless a physical prototype is built.
- Important correction: strict v2 checklist uses nine 4x4 posts. Earlier draft lists with six posts are superseded.

## Start here

Open documents/Chicken_Coop_Build_Spec_Master.pdf.

## Main folders

- documents/: master PDF plus markdown source documents.
- csv_checklists/: build spec, strict materials list, strict cut list, drawing requirements, Etsy image script, blockers, proof tracker.
- drawing_assets/: first real plan assets in PDF, PNG, and SVG.
- current_plan_assets/: current draft plan set, original 15 step cards, original infographic, and original material/cut CSVs.

## Approval gate

Do not publish the Etsy listing until the pre-publish blockers are cleared and the proof assets exist as original drawings, CAD screenshots, SketchUp views, prototype photos, or equivalent.
