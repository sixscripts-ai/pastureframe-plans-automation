# Brand Assets

Source files for logos, listing-image templates, and the PDF cover/footer
template live here. Final exports go alongside their products in
`products/[slug]/exports/`.

| File (to add) | Purpose |
|---------------|---------|
| `assets/brand/logo.svg` | Wordmark + pictogram. |
| `assets/brand/logo-mark.svg` | Square pictogram only. |
| `assets/brand/listing-image-templates.fig` | 10-slot Figma template. |
| `assets/brand/pdf-cover.fig` | PDF cover page template. |
| `assets/brand/pdf-footer.fig` | PDF footer with shop name + © year. |

These files are operator-managed. Keep large binaries out of Git via
`.gitignore` if they grow.
