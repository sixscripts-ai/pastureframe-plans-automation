# Final PDF Plan Package Outline From the 15 Step Cards

This outline turns the 15 image step cards into a final buyer-facing PDF plan package. The PDF should be presented as DIY digital plans, not engineered drawings or a safety guarantee.

## Recommended final PDF package structure

1. Cover page - product name, concept image, PastureFrame branding, and digital-plan disclosure.
2. Read-first safety page - DIY disclaimer, local-condition warning, towing warning, predator limitations, and build responsibility.
3. Design summary - footprint, capacities, mobility system, materials philosophy, and setup modes.
4. Master dimensions - top view, side view, section view, and key dimensions.
5. Materials list - stricter build-spec materials list.
6. Cut list - stricter build-spec cut list with field-fit notes.
7. Tool list and hardware checklist.
8. Step cards 1-15 expanded into build pages.
9. Drawing appendix - top, side, exploded frame, wheel/skid detail, nest/roost layout.
10. Broiler-mode setup sheet.
11. Layer-mode setup sheet.
12. Maintenance and pasture rotation sheet.
13. Pre-build checklist and final inspection checklist.
14. Buyer support and license page.

## Step-card expansion map

| Step | Step card | Final PDF page intent |
| --- | --- | --- |
| 1 | Build the base | Build the 10 ft x 10 ft skid base square, flat, and parallel. Install 4x6 skids, 2x6 perimeter rails, and 2x6 cross ties. |
| 2 | Add posts and frame | Install 4x4 post lines at the front, at the upper-coop front line, and at the rear. Brace before moving the frame. |
| 3 | Build the upper coop floor | Build the 4 ft x 10 ft elevated coop floor over the rear 48 in of the run using 2x6 framing and 3/4 in exterior plywood. |
| 4 | Frame the coop walls | Frame upper coop walls with a high front wall and lower rear wall to create roof slope. Include vents, cleanout, hatch, and pop door openings. |
| 5 | Add roof to coop | Install rafters, purlins, drip edge, and lightweight corrugated roofing with manufacturer-specified overlap and fasteners. |
| 6 | Sheath the coop | Install exterior plywood or T1-11, seal edges, and leave planned openings for ventilation, access doors, egg hatch, and pop door. |
| 7 | Install doors and windows | Build predator-resistant cleanout doors, nest access hatch, ventilation shutters/screens, lower run access door, and pop door. |
| 8 | Enclose the run | Install 1/2 in 19-ga galvanized hardware cloth on run walls, exposed run roof, vents, and optional predator apron. |
| 9 | Add wheels | Install skid + axle wheel system with wheels near the heavier coop end and a tow tongue at the opposite/front end. |
| 10 | Finish touches | Install nest boxes, roost bars, ramp, latches, feeders, waterers, paint/stain, caulk, and roof trim. |
| 11 | Move and rotate | Test towing on flat ground, move slowly, check wheel hardware, and rotate to fresh pasture before the run is overused. |
| 12 | Broiler setup | Use lower run as main grow-out space. Remove roosts/nest use, add extra feeders/waterers, and move frequently. |
| 13 | Layer setup | Use upper coop for nesting, roosting, night shelter, and egg collection. Keep nest boxes clean and roosts removable. |
| 14 | Daily checks | Check water, feed, latches, hardware cloth, shade, heat stress, bedding, predator damage, and egg collection daily. |
| 15 | Final use and maintenance | Inspect fasteners, wheels, roof, hardware cloth, latches, bedding, and mobility after the first week and monthly thereafter. |
